Resources directory is where you place supplementary files, to enrich your WURFL detection tool:

* Copy latest wurfl.xml file to this directory so you can use it in WURFL Update > Local file
* Copy latest device_pix folder to have the latest device images
* Copy no-device.png so you can have a default image for devices without an image.
